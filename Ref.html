Правовое уведомление

Данная система предназначена ТОЛЬКО для медицинских работников (педиатров, врачей скорой помощи, фельдшеров).

Расчёт доз лекарственных препаратов не является руководством к самолечению. Назначение и коррекцию лечения проводит только лечащий врач.

Система носит вспомогательный характер. Окончательное решение остаётся за специалистом.

KLAUS является личной архитектурной разработкой. Любое воспроизведение без разрешения автора запрещено.

⚠️ KLAUS PEDIATRIC v80 – CDSS на основе объективных данных. Все расчёты проверяйте по официальным источникам. НЕ заменяет врача.

❓ СПРАВКА ПО KLAUS PEDIATRIC v80

📌 О системе

KLAUS PEDIATRIC v80 — клинический калькулятор для педиатров, врачей скорой помощи и фельдшеров. Рассчитывает дозы лекарств, оценивает шкалы (Уэстли, PEWS, Глазго), показывает центильный график ВОЗ, выявляет критические состояния и лекарственные взаимодействия (DDI). Работает полностью офлайн после загрузки.

Новое в версии 80: ВЫБОР РЕЖИМА ДОЗИРОВАНИЯ ПРИ ОЖИРЕНИИ (реальный/скорректированный/идеальный вес), единый переключатель веса для лекарств и инфузий, красные флаги при ожирении, исправление ложного «стабильное» при отсутствии витальных данных, УЧЁТ СКФ ДЛЯ ЦЕФТРИАКСОНА, ЦЕФОТАКСИМА И АМОКСИЦИЛЛИНА.

⚙️ Ввод данных

Обязательно: возраст (1–18 лет) и вес (2–250 кг). Рекомендуется: рост, температура, ЧСС, ЧД, SpO₂, АД.

⚖️ Выбор режима веса при ожирении

Переключатель всегда виден. При ИМТ <85% — неактивен с пояснением. При ИМТ >85% — активен.

  • 🔴 РЕАЛЬНЫЙ вес — для шока, сепсиса, гипотензии, ИВЛ
  • 🟡 КОРРЕКЦИЯ — компромисс при ожирении: Ideal + 0.4×(Real-Ideal)
  • 🟢 ИДЕАЛЬНЫЙ вес — для плановой терапии, нефротоксичных препаратов

💊 Коррекция доз антибиотиков по СКФ

Система автоматически корректирует дозы следующих антибиотиков при снижении СКФ:

  • Гентамицин, Ванкомицин — СКФ <30: 50% дозы, СКФ 30–60: 75% дозы
  • Цефтриаксон, Цефотаксим — СКФ <30: 50% дозы
  • Амоксициллин — СКФ <30: 50% дозы + интервал 12 ч; СКФ 30–60: 75% дозы
  • Цефиксим — СКФ <30: 50% дозы, СКФ 30–60: 75% дозы

🧠 Адаптивный режим дозирования

При включении адаптивного режима дозы антибиотиков корректируются с учётом:

  • Шок (гипотензия) → +25% для всех антибиотиков (увеличенный объём распределения)
  • Дегидратация (Тали ≥3) → -30% для нефротоксичных (Гентамицин, Ванкомицин)
  • Печёночная недостаточность → -50% для макролидов (Азитромицин, Кларитромицин)

⚠️ Важное предупреждение

KLAUS — вспомогательный инструмент. Окончательное решение о лечении принимает лечащий врач. При ожирении выбор режима дозирования — ответственность врача.
